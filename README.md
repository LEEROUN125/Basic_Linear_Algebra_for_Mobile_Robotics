# Basic_Linear_Algebra_for_Mobile_Robotics

## Linear Algebra for Robotics
Welcome to the Linear Algebra for Robotics course! This repository contains a 2-week intensive program designed to bridge the gap between abstract linear algebra and practical mobile robotics (Land, Sea, and Space).

### 🚀 Overview
The goal of this course is to move beyond simple matrix calculations and understand linear algebra as the "language" of robot motion, control, and estimation.

Focus: Mobile Robotics (Thruster modeling, State estimation, Transformations)

Methodology: Watch (Lectures) → Code (Python) → Experiment (Visualization) → Internalize (Derivation)

### 📅 Course Roadmap
Week 1: Fundamentals & Systems
Day 01: Geometry of Linear Equations (Row vs. Column Picture)

Day 02: Matrix & Transformations (Coordinate Frames in Robotics)

Day 03: LU Decomposition & Elimination

Day 04: Transposes, Permutations, & Vector Spaces

Day 05: Column Space & Nullspace

Day 06: Orthogonality & Projection

Day 07: Least Squares (Robot Localization Intro)

Week 2: Eigen-world & Applications
Day 08: Eigenvalues & Eigenvectors

Day 09: Diagonalization & Differential Equations

Day 10: Singular Value Decomposition (SVD)

Day 11: SVD Applications in Robotics

Day 12: Advanced Transformations

Day 13: Numerical Methods

Day 14: Final Project: Mobile Robot Motion Planning
